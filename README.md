# GoogleMeet-Class

A Node.js + Express-based automation tool to join Google Meet classes on schedule using Puppeteer.

## Project Structure

```
src/
  configs/        # App configs and credentials
  controllers/    # Core logic
  models/         # Data models
  routes/         # Express routes
  schemas/        # Validation schemas
  app.js          # App setup
server.js         # App entry point
```

## Getting Started

1. Clone the repo
2. Run `npm install`
3. Add environment variables using `.env.example`
4. Start the server:

- `npm start`: (production)
- `npm run dev`: (development)
